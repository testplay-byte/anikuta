# Session Logs

One file per session, named `YYYY-MM-DD-session-N.md`.

## Template

```markdown
# Session Log — YYYY-MM-DD (Session N)

## Goal
<what we set out to do this session>

## What happened
- <step 1>
- <step 2>

## Decisions made
- <decision> — because <reason>

## Files touched
- <path> — <why>

## Next session
- <follow-up item>
```
