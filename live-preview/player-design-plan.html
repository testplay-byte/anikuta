Architecture: One Screen, Two Modes

The player is a single screen with two modes that the user can toggle between. The video keeps playing during the transition — no reload, no rebuffer.

Minimized Mode
  • Video at top (adapts to aspect ratio)
  • Episode info below video
  • Server + version (sub/dub) dropdowns
  • Scrollable episodes list
  • Tap video to show/hide controls
  • Swipe up → fullscreen
Fullscreen Mode
  • Video fills entire screen
  • Controls overlay on video (auto-hide)
  • All settings in top bar + bottom bar
  • Lock button (top left)
  • Gestures: seek, brightness, volume
  • Pinch zoom with magnetic snapping
  • Swipe down → minimized

Pinch Zoom — Magnetic Snapping

Pinch to zoom has magnetic sticky zones at every 20% interval. When zoom passes through a sticky point (100%, 80%, 60%, 40%), it "locks" there briefly. The user must zoom a bit extra to break past the sticky zone — this prevents accidental zoom changes and makes it easy to land on common zoom levels.

How it works: If the user is at 100% and zooms out, the zoom stays at 100% until they zoom out "hard enough". Once past the threshold, it drops to 80% and sticks there. Same pattern at every 20% mark. Also snaps when video edges align with screen edges (contain/cover fit).
Mode Transition

↕ Swipe Gestures

  • Swipe down from center → fullscreen to minimized
  • Video keeps playing during transition
  • Smooth animation (YouTube-style)

⛶ Button Toggle

  • Maximize button in minimized view → fullscreen
  • Minimize button in fullscreen → minimized
  • Same smooth animation as swipe
  • Video position preserved
Episode Switching

When the user taps an episode in the list (minimized mode):

  1. Current episode pauses
  2. Loading animation appears on the video player
  3. New episode loads in the background
  4. Video starts playing once loaded
  5. Episode info updates (title, description)
  6. List updates — new episode highlighted as "playing"

Settings Integration

A new "Player" section in Settings:

  • Default view — Minimized / Fullscreen / Ask every time
  • Default playback speed — 0.25x to 4x
  • Seek step — seconds per swipe (default 10)
  • Gestures enabled — toggle all gestures on/off
  • Pinch zoom enabled — toggle
  • Auto-hide controls — toggle + timeout
  • Skip button duration — customizable (default 85s)
  • Subtitle settings — font size, color, background, position (Phase 5)
