Improvement Plan (4 Phases)
Phase 1: Core Player UX
Essential
  • In-player quality selection — bottom sheet with available qualities
  • Subtitle track selection — bottom sheet listing tracks + "off"
  • Audio track selection — bottom sheet listing tracks
  • Playback speed — bottom sheet with speed slider (0.25x – 4x)
  • Settings menu — gear icon in top bar → opens a sheet with all options
Phase 2: Gesture Controls
Important
  • Horizontal swipe — seek forward/backward
  • Left vertical swipe — brightness
  • Right vertical swipe — volume
  • Double-tap — seek ±10s (with visual feedback triangles)
  • Pinch — zoom (optional)
Phase 3: Subtitle Customization
Enhancement
  • Subtitle settings sheet — font size, color, background, position, delay
  • Subtitle delay — sync adjustment (±ms)
  • Subtitle presets — "Small", "Medium", "Large", "Custom"
Phase 4: Polish
Future
  • PiP (Picture-in-Picture) — continue playback when switching apps
  • Media session — notification controls, headphone button support
  • AniSkip — auto-skip openings/endings
  • Sleep timer — stop after N minutes
  • Screenshot — capture current frame
  • Orientation lock — force landscape/portrait

Technical Notes

MPV Track API

MPV exposes tracks via the track-list property. Each track has: type (video/audio/sub), id, lang, title, default.

Select audio: MPVLib.setPropertyInt("aid", trackId)
Select subtitle: MPVLib.setPropertyInt("sid", trackId)

Subtitle Customization Properties

sub-scale — font size multiplier (1.0 = default)
sub-color — subtitle text color
sub-border-color — outline color
sub-back-color — background color
sub-pos — vertical position (0-100, 100 = bottom)
sub-border-size — outline thickness
sub-font — font family
sub-bold / sub-italic — font weight/style
sub-delay — timing offset (seconds)

All set via MPVLib.setProperty() — real-time, no restart needed.
